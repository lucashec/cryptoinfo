# CryptoInfo App

Aplicação para consulta de informações sobre criptomoedas com dados da API CoinkGecko API.
(https://www.coingecko.com/en/api)


## Novos aprendizados
- Consumo de API utilizando o axios
- Hooks de UseState e UseEffect
